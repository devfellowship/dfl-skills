# dfl-skills

DFL Skills — marketplace web app (`apps/web`) + install CLI (`packages/cli`). Consumes the `skills.devfellowship.com` API.

## Structure

```
apps/web/       — Vite + React skills marketplace UI (@dfl-skills/web)
packages/cli/   — @devfellowship/skills install CLI (find/add/install-mcp/install-connection)
```

## Getting Started

```bash
bun install
bun run typecheck
bun run build
bun run test
```
